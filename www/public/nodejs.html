node.js

2009 Ryan Dahl

시작하기

  • Node.js®는 크롬의 자바스크립트 런타임으로 만들어진 플랫폼
  • 확장성 있는 네트워크 애플리케이션을 쉽고 빠르게 제작 가능
  • 9만 개의 npm(Node Packaged Modules) http://npmjs.org

설치

  • http://nodejs.org OS별로 다운로드 받아서 설치
  • node --version 명령으로 설치 확인
  • 실행 안되는 경우 환경변수 Path 확인

node.js 코드

  • *.js 파일을 만들어서 실행
  • 간단 서버 예제 example.js
  • var http = require('http');
http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Hello World\n');
}).listen(1337, '127.0.0.1');
console.log('Server running at http://127.0.0.1:1337/');
  • 실행 방법: node example.js

npm 확장 모듈

  • npm install 모듈이름 node_modules/ 폴더에 해당 모듈 다운로드되어 require("모듈명")를 통해서 사용 가능
  • npm install -g 모듈이름 글로벌 모듈로 설치
  • npm install -g locally locally 라는 명령어 설치
  • locally --help 도움말
  • locally 실행 후 브라우저에서 http://localhost:8080/ 으로 접근 가능
