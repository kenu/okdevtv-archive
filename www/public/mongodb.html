MongoDB

2009 MongoDB, Inc.

개요

  • NoSQL (Not only SQL) 데이터베이스
  • JSON 형태로 저장
  • JavaScript로 제어
  • 오픈소스 라이선스(core:AGPL, driver:Apache License)

설치

  • OS에 맞는 mongodb 선택하고 다운로드, http://www.mongodb.org/downloads
  • 설치하고 bin폴더 환경변수 Path에 추가
  • cmd창에 mkdir /data/db 명령 실행
  • mongod 명령으로 몽고DB 데몬 실행

MongoDB 접속

  • cmd 창에서 mongo 명령 실행
  • show collections; 컬렉션(테이블) 목록 보기
  • db.콜렉션명.find(); 콜렉션 내용 보기
  • quit(); 클라이언트 종료

기본 컨셉

  • RDB와 다른 BSON(Binary JSON) 데이터 형태
  • {field: value, field: value, field: value}
  • MongoDB RDB
    document record
    collection table

데이터 입력

  • 컬렉션을 따로 만들 필요 없음
  • db.컬렉션.insert(object);
  • 예) db.testData.insert({name: "kenu"});
