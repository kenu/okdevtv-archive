# NPM
* Node Package Manager

## 설치
* node.js 와 함께 설치됨
* `package.json` 파일 참조

## 사용법

```
npm install 모듈명
npm i 모듈명
```

* `package.json` 모듈 추가

```
npm install --save 모듈명
```

* 글로벌 설치

```
npm i -g pm2
npm i -g express-generator
```

## npm 최초 시작

```
npm init
```

## npx
* npx(1) -- execute npm package binaries

```
npx -p node@8 npm run build
```
