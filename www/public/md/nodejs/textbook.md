## 예외 처리하기

* 예외 처리 못하면 Node 프로세스 멈춤. 예) 서버 다운
* 멀티 스레드 프로그램에서 하나가 멈추면 다른 스레드가 대신
* 싱글 스레드인 노드는 전체 서버가 멈춘다는 의미
* 예기치 못한 에러 처리하는 기술 필수

* 문법상의 에러 없다고 가정 - 실제 배포용 코드에 문법 에러가 있다는 것은 말이 안됨
* 좋은 에디터, 좋은 문법 검사 도구

```JS
setInterval(() => {
  console.log('시작');
  try {
    throw new Error('서버를 고장내주마!');
  } catch (err) {
    console.error(err);
  }
}, 1000);
```
