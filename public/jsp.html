JSP

1999 Sun Microsystems

시작하기

  • JavaServer Pages(JSP)
  • 게시판 같이 동적으로 HTML, xml 등 콘텐츠 생성
  • 톰캣, 제티 같은 서블릿 컨테이너 필요
  • 파일 확장자 jsp 사용

실행환경 톰캣

  • http://tomcat.apache.org/download-80.cgi 위치에서 Binary Distribution > Core > zip 링크로 다운로드
  • 적당한 위치(예 C:\java\apache-tomcat-8.0.x 또는 ~/java/apche-tomcat-8.0.x)에 압축 해제
  • JAVA_HOME 환경변수 필요
  • bin 폴더에서 startup.bat 실행
  • 창이 꺼지는 경우 conf/server.xml 에서 8080을 9090 등으로 포트 변경하고 저장한 후 다시 bin/startup.bat 실행
  • 브라우저에서 http://localhost:8080 으로 접속해서 설치 확인

JSP 디렉티브(Directives)

  • <%@ 디렉티브 %> 형식
  • page, taglib, include
  • 페이지에 필요한 자원 선언
  • <%@page import="java.util.*" %>
<%@page contentType="text/html;charset=utf-8" pageEncoding="utf-8" %>

JSP 표현식(Expressions)

  • <%= 표현식 %> 형식
  • 값 표현할 때 사용
  • <%= 3 * 4 * 5 %>
<%= new java.util.Date() %>

JSP 스크립틀릿(Scriptlets)

  • <% 스크립틀릿 %> 형식
  • 세미콜론(;)으로 끝나는 완전한 자바 문장
  • <%
    String hello = "Hello World";
    out.println(hello);
%>

JSP 선언자(Declaratives)

  • 권장하지 않는 기능
  • <%! 자바선언문 %> 형식
  • 메소드, 필드 변수 선언
  • 사용자 세션과 상관없이 공유되는 변수, 따라서 사용주의
  • out, request 등 JSP 내장객체 직접 사용불가

EL (Expression Language)

  • JSP 2.0(2003) 이후 사용가능
  • 표현식 대체 <%= hello %> 대신 ${hello} 사용
  • <%= request.getParameter("hello") %> 대신 ${param.hello}
