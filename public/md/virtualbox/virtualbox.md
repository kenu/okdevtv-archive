# VirtualBox
* 오픈 소스 가상머신 by Oracle
* http://www.virtualbox.org
* 윈도우, 리눅스 등의 OS를 멀티부팅하지 않고 사용 가능

## 용어
* VM
  * Virtual Machine, 가상 머신, Host OS 위에 설치되는 Guest OS
* Host OS
  * VirtualBox 프로그램을 설치하는 OS
* Guest OS
  * VirutalBox 프로그램 내에 설치되는 가상 OS
