# go lang
* https://golang.org/


## Download & install
* https://golang.org/dl/

### for mac
* `brew install go`
* `go env`

## hello world

* hello.go

```
package main

import "fmt"

func main() {
	fmt.Println("Hello, 世界")
}
```

* 실행

```
go run hello.go
```

* build

```
go build hello.go
./hello
```
