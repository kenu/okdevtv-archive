# zeppelin
* https://zeppelin.apache.org
* Multi-purpose notebook
  * Data Ingestion
  * Data Discovery
  * Data Analytics
  * Data Visualization & Collaboration
  
## Start, Stop
```
bin/zeppelin-daemon.sh start
bin/zeppelin-daemon.sh stop
```
