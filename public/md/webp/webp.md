# WebP
* Optimized picture format by google
* https://developers.google.com/speed/webp/

## install
* `brew install webp`

## Usage
* basic
```
cwebp filename.png -o filename.webp
cwebp -q 80 filename.png -o filename.webp
```

* resize
```
cwebp -q 80 -resize 440 276 filename.png -o filename.webp
```
