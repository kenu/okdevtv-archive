MEAN.io

from: http://mean.io

여러분의 차세대 웹 애플리케이션을 위한 친철하고, 재미있는 자바스크립트 풀스택

MEAN은 고집스럽게 추천하는 풀스택 자바스크립트 프레임워크입니다 - 단순하게 가속시키는 개발을 가능하게 합니다.

MEAN을 실행하는 법...

mac || linux

$ sudo npm install -g mean-cli
$ mean init yourNewApp

win

$ npm install -g mean-cli
$ mean init yourNewApp

MongoDB

MongoDB는 앞서가는 NoSQL 데이터베이스이고, 비즈니스가 더 애자일하고 확장성있게 만들어 줍니다.

Express

Express는 작고, 유연한 node.js 웹 애플리케이션 프레임워크입니다. 하나 이상의 웹 페이지, 또는 하이브리드 웹 애플리케이션 제작을 위한 견고한 기능들을 제곰합니다.

AngularJS

AngularJS lets you extend HTML vocabulary for your application. The resulting environment is extraordinarily expressive, readable, and quick to develop. AngularJS로 여러분은 HTML 어휘를 확장할 수 있습니다. 궁극의 환경은 매우 표현적이고, 가독성이 높으며, 개발을 빨리 할 수 있습니다.

NodeJS

Node.js는 크롬 브라우저의 자바스크립트 런타일 기반의 플랫폼입니다. 네트워크 애플리케이션을 만드는데, 쉽고, 빠르고, 확장성이 있습니다.
