  • 콘텐츠는 태그 사이에 위치
  • 웹 애플리케이션을 위한 HTML5 표준 2014 Q4 완성 예정
  • 스타일은 CSS, 액션은 JavaScript에게 위임

페이지 소스 기본 구조

  • <!DOCTYPE html>
  • <html>
    • <head>
    • <body>

head 태그

  • 콘텐츠 이외의 정보
  • title, meta, link, script

body 태그

  • 콘텐츠 표시 영역

문법

  • <태그 속성="값">콘텐츠</태그>
